# Nutrition Planner

## Goal
Build a lightweight wellness product that turns nutrition goals into actionable meal plans, grocery lists, and weekly tracking.

## Product surface
- user-facing app
- simple dashboard
- weekly planning workflow

## Inputs
- goal
- dietary preferences
- pantry state
- grocery budget

## Outputs
- weekly nutrition plan
- grocery list
- adherence summary

## Ecosystem expectations
- consume agentic-os task and artifact contracts
- register artifacts into graphify
- allow Bttr to consume accountability events
